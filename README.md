# FCA-iTransformer 设计文档

## 1. 项目概述
- **目标**：预测粮库面粉重量随温度和湿度变化的多变量时间序列。
- **创新点**：在 iTransformer 基础上增加 **频率增强信道注意力（FCA）** 模块。
- **功能**：
  - 高频增强：捕捉局部波动
  - 通道注意力：自适应加权各变量

## 2. 模型结构
### 数据流
原始序列 X → FCA → 嵌入层 → iTransformer 编码器 → 预测输出 → Y_hat

### 模块说明
1. **FCA模块**：频率增强 + 通道注意力  
2. **嵌入层**：提取时间与变量特征  
3. **iTransformer编码器**：多层自注意力机制建模时间和变量依赖  
4. **预测输出层**：输出未来 4 小时重量  

## 3. 模块特点
- 高频增强：敏感识别局部波动  
- 通道注意力：关键变量加权，提高多变量预测精度  
- 模型通用性强，可用于其他多变量时间序列预测任务




# FCA-iTransformer 使用说明文档

## 1. 数据集
- **特征**：温度（℃）、湿度（%RH）、重量（kg）  
- **采样频率**：每小时  
- **数据划分**：
  - 训练集：181 条  
  - 验证集：26 条  
  - 测试集：54 条  

## 2. 实验配置
| 参数 | 值 |
|------|----|
| seq_len | 16 |
| pred_len | 4 |
| 特征维度 | 3 |
| d_model | 512 |
| d_ff | 2048 |
| 编码器层数 | 2 |
| 注意力头 | 8 |
| Dropout | 0.2 |
| Batch Size | 32 |
| 学习率 | 0.0001 |
| 训练轮次 | 50 |
| 提前停止 | 10 |

## 3. 实验结果
| 模型 | MSE | MAE |
|------|------|------|
| DLinear | 0.1629 | 0.2892 |
| FEDformer | 0.1344 | 0.2701 |
| Autoformer | 0.2410 | 0.3468 |
| iTransformer | 0.1100 | 0.2177 |
| **FCA-iTransformer** | **0.1024** | **0.2087** |

**结论**：FCA-iTransformer 在测试集上预测精度最高，局部波动拟合更准确。
